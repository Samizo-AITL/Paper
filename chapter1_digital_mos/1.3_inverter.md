# 1.3: CMOSインバータの構成と動作

## ✅ CMOSインバータとは

CMOS（Complementary MOS）は、nMOSとpMOSを組み合わせた基本構造です。

- 入力が1 → pMOS OFF, nMOS ON → 出力は0
- 入力が0 → pMOS ON, nMOS OFF → 出力は1

## ✅ 回路図（テキスト表現）
```
Vdd
|
pMOS
|
+——→ 出力
|
nMOS
|
GND
```

## ✅ 入力-出力の関係（波形で表現）

| 入力 | 出力 |
|------|------|
| 0    | 1    |
| 1    | 0    |

🧠 ポイント：CMOSは消費電力が少なく、ノイズ耐性が高い構造。

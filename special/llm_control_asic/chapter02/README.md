# 第2章 基礎知識

## この章の目的

この章では、LLM（大規模言語モデル）、制御理論、FPGA/ASIC設計に関する基礎的な知識を解説します。  
本教材の後続章を理解する上で必要となる「技術の土台」を固めることが目的です。

## なぜこの章が必要か？

- LLMと制御理論はまったく異なる前提を持つ技術領域であるため、共通理解が必要  
- FPGAやASICの設計フローに不慣れな場合、実装面の理解が難しくなる  
- この章で全体の「技術語彙」と「前提知識」を統一しておくことで、後の章の理解がスムーズになる  

## 本章で扱う内容

- LLMの基礎（構造、学習方法、応用領域）  
- 制御理論の基本（PID制御、ロバスト制御、MPCなど）  
- ASIC設計の基本フローと用語  
- FPGAの構造・開発手順、ASICとの比較

## 本章を学ぶと何が得られるか？

- LLMの推論原理と、制御理論の設計思想の違いと補完関係を理解できる  
- 制御設計と機械学習の融合設計における基盤知識を習得できる  
- FPGAとASICの設計プロセスの全体像が把握できる  
- 実装作業における用語・工程・技術選択の判断力が向上する

## 推奨する読み方

- 制御系の知識がある読者は、LLMやASIC部分を重点的に  
- AI系の読者は、制御理論とデジタル回路の項目を丁寧に  
- 全体の理解が必要な開発リーダー層は、章全体を通読推奨

---

# 📘 edusemi v2.0 – 半導体基礎教育資料

**edusemi** は、半導体の基礎から実践的設計・実装技術に至るまでを体系的に学べる、オープンな教材群です。デバイス物理、プロセス技術、SoC設計、パッケージ・テスト、さらに開発レビューまで、**教育的なわかりやすさ**と**実務的な深さ**を両立し、以下のような幅広い層に活用されています：

- 初学者（大学・高専生、高校生）
- 新人エンジニア、若手技術者
- 社会人のリスキリング
- 教育機関・企業研修の講師
